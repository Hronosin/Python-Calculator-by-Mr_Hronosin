# 🚀 Інженерний калькулятор — Улучшенна версія

## 🎨 Дизайн: Мінімалістичний космос

Новий інтерфейс створений в стилі мінімалістичного космосу з:
- **Темна палітра**: глибокий космічний фон (#0a0e27)
- **Неон-забарвлення**: 
  - Циан (#00d9ff) — основний колір
  - Фіолетовий (#b026ff) — оператори
  - Рожевий (#ff006e) — дії
  - Зелений (#00ff88) — успіх

## 🎯 Основні можливості

### Редагування виразу в будь-якому місці
- Курсор можна розташовувати **посередині** виразу
- **DEL** видаляє символ перед курсором
- **Ctrl+A** виділяє весь текст
- Стирання одного символу — не потрібно перепісувати весь вираз

### 📊 Математичні функції
- Тригонометричні: `sin`, `cos`, `tan`, `asin`, `acos`, `atan`
- Гіперболічні: `sinh`, `cosh`, `tanh`
- Логарифми: `log`, `ln`, `log10`
- Інші: `sqrt`, `root`, `exp`, `factorial`, `abs`
- Комплексні числа: введіть `i` або `2+3j`
- Константи: `pi`, `e`

### 📈 Графіки
- Клік **Plot** для побудови графіка функції
- Потрібна змінна `x` в виразі
- Наприклад: `sin(x)`, `x^2-2*x+1`, `1/x`

### 🧮 Автоматизація розрахунків
- **d/dx** — чисельна похідна в точці
- **∫** — чисельний визначений інтеграл
- Введіть межи формату `0, 5` для інтегралу

### 🔄 Конвертація одиниць
- **Довжина**: `km_to_m`, `m_to_km`, `mi_to_km`, `km_to_mi`
- **Температура**: `c_to_f`, `f_to_c`, `c_to_k`, `k_to_c`
- **Кути**: `deg_to_rad`, `rad_to_deg`
- Приклади: `c_to_f(100)`, `km_to_m(5)`, `deg_to_rad(180)`

### 📋 Історія обчислень
- Всі вирази зберігаються автоматично
- **Подвійний клік** на записі — вставити вираз назад
- **Import/Export** як JSON
- **Clear** — очистити історію

### 🌍 Локалізація
Мени мова:
- 🇬🇧 English
- 🇺🇦 Українська
- 🇷🇺 Русский

## 💡 Приклади виразів

```
2 + 2 * 3              → 8
sin(pi/2)              → 1.0
sqrt(-1)               → 1j (комплексне число)
2+3j                   → (2+3j)
factorial(5)           → 120
log(100)               → 4.605... (натуральний логарифм)
log10(100)             → 2.0 (логарифм за основою 10)
deg_to_rad(180)        → 3.14159... (π радіан)
c_to_f(0)              → 32.0 (температура)
```

## 🖱️ Користування

1. **Введіть вираз** у поле вводу
2. **Натисніть Enter** або клік кнопку `=`
3. **Результат** з'явиться у полі вводу
4. **Ctrl+C** щоб скопіювати
5. **Подвійний клік** на історії щоб відновити вираз

## 📝 Примітки

- Поле вводу повністю редаговано — можна редагувати хоч де
- Статус-бар внизу показує підказки та повідомлення
- При помилці — червоне повідомлення в статус-барі
- При успіху — зелене повідомлення зникне через 2 сек
