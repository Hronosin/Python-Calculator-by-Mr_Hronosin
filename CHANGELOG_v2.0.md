## ⌨️ Редактирование текста [НОВОЕ]

### Раньше
```
Выражение: 2+2*3
Ошибка в начале → приходилось стирать ВСЁ
```

### Сейчас
```
Курсор в любом месте → редактируй как обычный текст
Ctrl+A → выделить всё
Delete → удалить символ перед курсором
Вставь функцию в середину → просто!
```

### Реализация
```python
pos = self.display.index("insert")  # Получить позицию курсора
self.expression.set(current[:pos] + value + current[pos:])  # Вставить в позицию
self.display.icursor(pos + len(value))  # Переместить курсор
```

## 🧬 Математические функции [РАСШИРЕНО]

### Новые функции
| Категория | Функции |
|-----------|---------|
| Обратные триго | asin, acos, atan |
| Гиперболические | sinh, cosh, tanh |
| Логарифмы | log(base e), log10, ln |
| Другое | exp, root, abs |
| Комплексные | Поддержка i и 2+3j |

### Примеры
```
asin(1)        → 1.5708... (π/2)
sinh(1)        → 1.1752...
exp(2)         → 7.3891...
root(27, 3)    → 3.0 (кубический корень)
sqrt(-1)       → 1j (комплексное число)
2 + 3j         → (2+3j)
```

## 📊 Новые инструменты [ДОБАВЛЕНО]

### Графики функций
```python
def plot_expression(self):
    # Строит график для выражений с x
    # Диапазон: -10 to 10
    # Масштаб автоматический
    # Осями помечены координаты
```

**Примеры**: `sin(x)`, `x^2-2*x+1`, `1/(x+1)`

### Производные (численно)
```python
h = 1e-6
derivative = (f(x+h) - f(x-h)) / (2*h)
```

**Использование**: Клик d/dx → ввести x → получить результат

### Интегралы (численно)
```python
# Метод трапеций
step = (end - start) / steps
for i in range(steps):
    integral += (f[i] + f[i+1]) * step / 2
```

**Использование**: Клик ∫ → ввести "a, b" → получить результат

### Конвертация единиц
```python
# Длина
km_to_m(x)      # км → м
mi_to_km(x)     # мили → км

# Температура  
c_to_f(x)       # °C → °F
c_to_k(x)       # °C → K

# Углы
deg_to_rad(x)   # градусы → радианы
```
